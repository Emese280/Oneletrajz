Szoftverfejlesztő és tesztelő technikus

SZC Széchenyi István Műszaki Technikum szoftverfejlesztő és tesztelő szakán kezdtem tanulmányaimat, ahol már az első évben megtanultam Pythonban programozni.

Elsajátítottam a C#/.NET technológiát, az objektumorientált programozást és az algoritmikus gondolkodást, erős hangsúlyt fektetve a Clean Code elveire.

Diákmunkában sírhelyek adatait digitalizáltuk, majd feltérképeztük a területet községünk számára.

Konzolosan és grafikusan is fejlesztettem alkalmazásokat OOP szemlélettel, valamint backend és frontend alapú, adatbázisos MVC rendszereket is készítettem.

Tovább mélyítettem szakmai tudásomat projektalapú fejlesztésekben.

Vizsgaremek

MVC technológiával készítettem egy időpontfoglaló webalkalmazást, amelyen keresztül backend és frontend tapasztalatot is szereztem.
